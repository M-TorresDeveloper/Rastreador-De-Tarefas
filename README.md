# Rastreador De Tarefas

App Ratreador De Tarefas(Task Tracker) criado com React JS.

## Uso

### Instalar dependencias

```
npm install
```

### Rodar no servidor (http://localhost:3000)

```
npm start
```

### Rodar no JSON server (http://localhost:5000)

```
npm run server
```

### Build para produção

```
npm run build
```
